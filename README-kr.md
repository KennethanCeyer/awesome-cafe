- `개발자 카페`의 조건은 아래와 같습니다.
  - 카페에서 앉을 수 있는 좌석이 제공 될 것.
  - 카페에 콘센트가 존재할 것.
  - `4시간` 이상 이용이 가능할 것.
  - 와이파이가 제공 될 것.
  - 1인이 앉을 수 있을 것.
